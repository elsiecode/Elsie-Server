Elsie v1.0.0.

How To Use

1.Open the elsie folder

index.php file. Once Elsie is uploaded to the server this file launches the functions needed to build your webpage. It will also let you know if an index.html file (website homepage) already exsits on the server.

User folder. This folder holds the text files Elsie will read to decide how to build your webpage.

ElsieFiles folder. This folder holds all the task functions Elsie needs to build your webpage.

2.Create content

Elsie uses file names to identify the type of content a text file may contain. Elsie accepts three text files. site-info.txt for general site infomation, blog-post.txt for post content and style.txt for styling the webpage.

Elsie uses tags within a text file to identify how the content should be placed into the webpage. Elsie tags look like this. <open tag>Some content here</close tag>

You can learn more about which tags are accepted in a text file by reading below.

Site information

To add site information create a new file name it "site-info.txt".

site-info.txt accepts the following tags

<title>Name of your webpage</title>

<description>One or two lines about your webpage.</description>

Once finished save inside User folder.

Creating a coming soon page

To create a generic coming soon page with the message "Website Coming Soon" leave User folder empty.

To create a coming soon page with the title of your webpage create a file name it "site-info.txt" type inside the file <title>Name of your webpage</title> and save it to the User folder.

Creating a blog post

To create a blog post create a new file name it "blog-post.txt".

blog-post.txt accepts the following tags

<title>Title of post</title>

<heading>Paragraph title</heading>

<paragraph>Paragraph text</paragraph>

<link><url>example.com</url><text>Link Text</text></link>

Once finished save inside User folder.

3.Style content

To style content create a new file name it "style.txt".

style.txt accepts the following tag

<theme>Theme name</theme>

Available themes nightsky, grasshopper, lights out.

For white background with black text it is not necessary to specify a theme.

Once finished save inside User folder.

4.Upload Elsie

Once finished creating the text files and saving them to the User folder you will need to upload the elsie folder to your web server.

Please note that Elsie is indented to be used at the time of upload. Do not upload the elsie folder until you a ready to create your webpage.

Be sure to place the elise folder in the same folder that holds public webpages for your site.

Go to yourdomain/elsie. If you get an error make sure you uploaded Elsie to the right location and that your webhost has php installed on the server (This is typically already installed when activating your hosting account.)

When finished you should see your completed webpage at which time the elsie folder will remove itself from the server. It is recommended that you check the file manager to make sure Elsie is completely removed from server.

Development

Notes

Elsie outputs the webpage in standard html and css. If you wish to expand upon what Elsie built all you need to do is download the index.html file from your server and open it in a text/code editor.

Elsie Varibales

All Elsie varibales use the prefix "e_" to add your own varibles avoid using this prefix.

Elsie Functions

All Elsie functions use the prefix "elsie_" to add your own functions avoid using this prefix.
